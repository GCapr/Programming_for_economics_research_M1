Course Research Project

Throughout this course, we build a complete research project: "Climate Vulnerability and Economic Growth." This module begins gathering the data we'll analyze.

What is Data Harnessing?

Data harnessing is the process of acquiring, exploring, and preparing data for analysis. Before you can test hypotheses or build models, you need to:

  • Find and access the right data sources
  • Understand what your data contains and its quality
  • Transform raw data into analysis-ready formats

This is often the most time-consuming part of research—expect to spend 60-80% of project time on data work!

Two Paths to Data

Researchers typically acquire data through two main channels, each with different skills and challenges:

Path A: Static Files

Data that has already been collected and stored in files (CSV, Excel, Stata .dta)

  • Survey datasets
  • Administrative records
  • Published research data
  • Downloaded extracts

Key skills: Loading, inspecting, summarizing, visualizing

Path B: Live Sources (APIs)

Data accessed programmatically from online databases and services

  • World Bank indicators
  • Census Bureau data
  • Economic databases (FRED)
  • Social media data

Key skills: HTTP requests, authentication, JSON parsing

Module Structure

This module is split into two subpages to give each path the attention it deserves:

Part 2a: Data Exploration →

Start here if you have data files ready to analyze. Learn the essential "First Analysis" workflow:

  • Loading data from CSV, Excel, and Stata formats
  • Inspecting structure and identifying data types
  • Computing summary statistics and detecting issues
  • Creating visualizations to understand distributions
Part 2b: Working with APIs →

Move here when you need to fetch data programmatically. Learn to:

  • Understand what APIs are and how they work
  • Make HTTP requests to retrieve data
  • Work with JSON responses
  • Handle authentication and API keys
