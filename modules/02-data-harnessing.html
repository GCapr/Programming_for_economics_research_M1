Quick Reference: Loading Data

Real-world data comes in many formats. The most common are CSV, Excel spreadsheets, and Stata's .dta files.

# Python: Loading data with pandas
import pandas as pd

# CSV
df = pd.read_csv("data.csv")

# Excel
df = pd.read_excel("data.xlsx", sheet_name="Sheet1")

# Stata
df = pd.read_stata("data.dta")

# From URL
url = "https://example.com/data.csv"
df = pd.read_csv(url)
* Stata: Loading data

* CSV
import delimited "data.csv", clear

* Excel
import excel "data.xlsx", firstrow clear

* Stata
use "data.dta", clear

* From URL
use "https://example.com/data.dta", clear
# R: Loading data
library(tidyverse)
library(haven)
library(readxl)

# CSV
df <- read_csv("data.csv")

# Excel
df <- read_excel("data.xlsx")

# Stata
df <- read_dta("data.dta")

2.2 Merging Datasets

Merging combines two datasets based on common key variables. There are four main types: inner, left, right, and outer joins.

# Python: Merging with pandas
merged = pd.merge(df1, df2, on="id")                   # Inner join
merged = pd.merge(df1, df2, on="id", how="left")       # Left join
merged = pd.merge(df1, df2, on=["id", "year"])         # Multiple keys
* Stata: Merging datasets
merge 1:1 id using "other_data.dta"
tab _merge
# R: Merging with dplyr
merged <- inner_join(df1, df2, by = "id")
merged <- left_join(df1, df2, by = c("id", "year"))

2.3 Reshaping Data

Data comes in wide format (one row per unit) or long format (multiple rows per unit). Most analysis requires long format.

# Python: Reshape wide to long
long = pd.melt(wide, id_vars=["country"], var_name="year", value_name="gdp")

# Long to wide
wide = long.pivot(index="country", columns="year", values="gdp")
* Stata: Reshape
reshape long gdp, i(country) j(year)
reshape wide gdp, i(country) j(year)
# R: Reshape with tidyr
long <- wide %>% pivot_longer(cols = starts_with("gdp"), names_to = "year")
wide <- long %>% pivot_wider(names_from = year, values_from = gdp)
