Visual Studio Code Guide

~45 min For Python users

Visual Studio Code (VS Code) is a free, powerful code editor that has become the most popular IDE in the world. It works with virtually any programming language and is my recommended editor for Python development.

Why VS Code?

If you're going to learn Python, I recommend using VS Code for several reasons:

  • Free and open-source — No cost, no licensing hassles
  • Works with everything — Python, R, Stata syntax, Markdown, LaTeX, and more
  • Excellent Python support — With the Python extension, it rivals dedicated Python IDEs
  • Git integration — Version control built right in
  • AI coding assistants — GitHub Copilot and Claude Code integrate seamlessly
  • Huge extension ecosystem — Add functionality for any use case

Installing VS Code

  1. Download from code.visualstudio.com
  2. Run the installer (works on Windows, Mac, and Linux)
  3. Launch VS Code

The VS Code Interface

VS Code has a clean, customizable interface. Here are the main components:

The VS Code interface. Image credit: Microsoft VS Code Documentation.

Interface Components

Activity Bar (Far Left)

The vertical bar with icons for:
Explorer — Browse files in your project
Search — Find text across all files
Source Control — Git integration
Run & Debug — Debugging tools
Extensions — Install new functionality

Side Bar (Left)

Shows the content for whichever Activity Bar icon is selected. Usually you'll have the Explorer open, showing your project's file tree. Click on a file to open it in the editor.

Editor (Center)

The main area where you write code. You can open multiple files in tabs, split the editor to view files side by side, and even preview Markdown files. VS Code provides syntax highlighting, error detection, and autocomplete.

Panel (Bottom)

Contains the Terminal (run commands without leaving VS Code), Problems (see errors and warnings), Output (logs from extensions), and Debug Console. Toggle it with Ctrl+`.

Installing Extensions

VS Code's power comes from extensions. Here's how to install them:

  1. Click the Extensions icon in the Activity Bar (or press Ctrl+Shift+X)
  2. Search for the extension name
  3. Click Install

Essential Extensions for This Course

Extension What it does Required for
Python (by Microsoft) Python language support, IntelliSense, debugging Python development
Jupyter Run Jupyter notebooks inside VS Code Interactive Python
Pylance Fast, feature-rich language support for Python Better Python autocomplete
GitLens Enhanced Git integration and history viewing Version control

Start with Python Extension

At minimum, install the Python extension by Microsoft. It will automatically recommend Pylance as well. These two extensions transform VS Code into a powerful Python IDE.

Setting Up Python in VS Code

After installing the Python extension, you need to configure VS Code to use your Python installation:

  1. Open VS Code
  2. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac) to open the Command Palette
  3. Type "Python: Select Interpreter" and press Enter
  4. Choose your Python installation from the list

If you don't have Python installed, download it from python.org or use Anaconda from anaconda.com.

Running Python Code

There are several ways to run Python in VS Code:

Method 1: Run Entire File

  • Open a .py file
  • Click the Play button (top right) or press Ctrl+F5
  • Output appears in the Terminal panel

Method 2: Run Selection (Interactive)

  • Select some code
  • Press Shift+Enter
  • VS Code runs the selection in an interactive Python window

Method 3: Jupyter Notebooks

  • Create a new file with .ipynb extension
  • Write code in cells and run them individually
  • Great for exploratory analysis (see the Notebooks guide)

The Integrated Terminal

One of VS Code's best features is the integrated terminal. Open it with Ctrl+` (backtick). You can:

  • Run Python scripts: python my_script.py
  • Install packages: pip install pandas
  • Use Git commands: git status
  • Navigate directories without leaving VS Code

Using AI Coding Assistants

VS Code integrates well with AI coding tools, which can dramatically accelerate your learning and productivity.

GitHub Copilot

GitHub Copilot provides real-time code suggestions as you type. It's trained on billions of lines of code and can complete entire functions from comments.

  • Cost: $10/month, but free for students (apply at education.github.com)
  • Install: Search "GitHub Copilot" in Extensions

Claude Code

Claude Code is Anthropic's AI coding assistant. There are two ways to use it:

Option 1: Claude Extension

  1. Open Extensions (Ctrl+Shift+X)
  2. Search for "Claude" or "Anthropic"
  3. Install the official extension
  4. Sign in with your Anthropic account

Option 2: Claude Code CLI in Terminal

You can use the Claude Code command-line tool directly in VS Code's integrated terminal: 

# Install Claude Code globally
npm install -g @anthropic-ai/claude-code

# Navigate to your project
cd /path/to/your/project

# Start Claude Code
claude

# Now you can ask questions like:
> "Explain what this file does"
> "Fix the bug on line 42"
> "Write tests for my function"
> "Convert this R code to Python"
